Customer Transactions Template
Template untuk import data transaksi pelanggan ke sistem CRM
Kolom yang Diperlukan:
  • unique_customer_id - ID unik pelanggan
  • customer_name - Nama lengkap pelanggan
  • customer_email - Email valid
  • customer_phone - Nomor telepon
  • product_type - Kategori produk
  • product_name - Nama produk
  • quantity - Jumlah pembelian
  • unit_price - Harga per unit
  • total_amount - Total transaksi
  • transaction_date - Tanggal transaksi
  • payment_method - Metode pembayaran
Format Penting:
  • Tanggal: YYYY-MM-DD HH:MM:SS (contoh: 2024-01-15 10:30:00)
  • Email: Format valid (contoh@email.com)
  • Harga: Angka murni tanpa titik/koma (contoh: 8500000)
  • Payment Method: credit_card, bank_transfer, cash, e_wallet
  • Jangan ubah nama kolom header!
Cara Menggunakan Template
Langkah-langkah:
1
Download Template
Klik tombol download untuk mendapatkan file CSV template
2
Buka dengan Excel/Spreadsheet
Buka file dengan Microsoft Excel, Google Sheets, atau LibreOffice Calc
3
Isi Data Anda
Ganti data contoh dengan data real Anda, jangan ubah header kolom
4
Save as CSV
Simpan file dalam format CSV (UTF-8) untuk kompatibilitas terbaik
5
Upload ke Import Data
Kembali ke halaman Import Data dan upload file CSV Anda
Tips & Peringatan:
Tips:
  • Validasi data sebelum upload
  • Cek format tanggal dan angka
  • Pastikan email valid
  • Test dengan data kecil dulu
Peringatan:
  • Jangan ubah nama kolom header
  • Harga harus berupa angka murni
  • Format tanggal: YYYY-MM-DD HH:MM:SS
  • File maksimal 5MB
Hasil yang Diharapkan:
  • Data ter-import dengan benar
  • Statistik terupdate otomatis
  • History import tersimpan
  • Laporan hasil import tersedia
